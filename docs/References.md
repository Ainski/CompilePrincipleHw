# 参考资料

> 本项目开发过程中查阅的主要参考资料，涵盖 Rust 语言规范、编译原理教材、同类编译器实现以及第三方库文档。

## Rust 语言与工具链

1. **[Rust 程序设计语言（中文版）](https://rustwiki.org/zh-CN/book/)** — Rust 官方入门教程，涵盖所有权、生命周期等核心概念。
2. **[Cargo 教程 | 菜鸟教程](https://www.runoob.com/rust/cargo-tutorial.html)** — Rust 包管理与构建工具 Cargo 的快速入门。

## 编译原理

3. **[自己动手写编译器](https://pandolia.net/tinyc/index.html)** — 以 TinyC 为例，从零实现一个 C 语言编译器的中文教程。
4. **[北京大学编译实践课程在线文档](https://pku-minic.github.io/online-doc/)** — 北大编译实践课程的实验文档，包含词法分析与语法分析的详细实现指导。

## 语法参考

5. **[grammars-v4/rust (ANTLR)](https://github.com/antlr/grammars-v4/tree/master/rust)** — ANTLR 社区维护的 Rust 语法规则文件，可作为语法设计参考。

## 第三方库

6. **[CLI11](https://github.com/CLIUtils/CLI11)** — C++11 命令行参数解析库，用于本项目 CLI 模式的参数处理。
7. **[Dear ImGui (Allegro5 示例)](https://github.com/ocornut/imgui/blob/master/examples/example_allegro5/main.cpp)** — ImGui 与 Allegro5 集成的参考实现，用于本项目 GUI 模式的搭建。
